# Student Management System (DBMS Project)

This is a beginner DBMS project that stores and manages student academic records.

## Entities
- Student
- Course
- Enrollment

## Features
- Store student details
- Store course information
- Track student enrollments
- Record grades

## Technologies Used
SQL / MySQL

## Database Schema
Tables:
1. Student
2. Course
3. Enrollment
